Privacidad

Esta maqueta almacena datos demo en el navegador para simular reservas y fidelización mientras se prepara el nuevo proyecto de Firebase de Elegance.

Datos demo

  • Reservas, perfiles y sellos se guardan localmente en `localStorage`.
  • No se usan credenciales reales del cliente en esta versión.
  • Las reglas productivas deben desplegarse desde `firestore.rules` cuando exista el proyecto final.
